# pw

pw is a really simple password manager.

## Usage

```sh
./pw <options>
```

| Option | Description |
| ------ | ----------- |
| -g \<password length\> | Generate a password containing numbers, letters and special characters. |
| -c \<password\> | Check a password's stength. |
